# Features

- Basic methods implemented
- 2 java code pages

## Methods

#### getCurrentSize()
- Returns the current size of bag
#### isEmpty()
- Returns the boolean value true if it is empty
#### add()
- Adds items in the baf and if it is succes returns true
#### remove()
- Removes the item from the bag and it returns bool depends to the succes of operation
#### clear()
- Removes all elements of bag
#### contains()
- Checks if item in the bag and returns bool
#### toArray()
- it returns the bag as array to outside of the adt
